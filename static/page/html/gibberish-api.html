Gibberish API Documentation - Frenz

Welcome to the Gibberish API documentation. You can freely use my API, enjoy!

Generate Gibberish

Generate gibberish for a given word.

GET https://frenzvalios.xyz/api/gibberish/generate?word=original_word

Parameters:

  • word (string): The original word to generate gibberish for.

Response:

{ "original": "original_word", "gibberish": "generated_gibberish" }

Translate Gibberish

Translate gibberish back to the original word.

POST https://frenzvalios.xyz/api/gibberish/translate?gibberish=gibberish_word

Parameters:

  • gibberish (string): The gibberish word to translate.

Response:

{ "original": "original_word", "gibberish": "gibberish_word" }
