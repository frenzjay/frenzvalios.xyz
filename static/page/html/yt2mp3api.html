API:

This API allows you to fetch audio URLs from YouTube videos.

Usage

To use the API, make a GET request to the following endpoint:

https://frenzvalios.xyz/api/yt2mp3?url=YOUR_YOUTUBE_VIDEO_URL

Parameters

The url parameter is required and should contain the YouTube video URL for which you want to fetch the audio URL.

Example

Suppose you want to fetch the audio URL for a YouTube video. Replace YOUR_YOUTUBE_VIDEO_URL with the actual YouTube video URL:

/youtube?url=YOUR_YOUTUBE_VIDEO_URL

Response

The API will respond with a JSON object containing the audio URL:


{
    "audio_url": "https://audio.example.com/audio12345"
}

If the request is successful, you can use the provided audio_url to play the audio.
